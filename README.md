# Login Page Automation Testing (Basic)

##  Project Description
This project automates the testing of a login page using Selenium and Python.

##  Features
- Automated login functionality testing
- Valid & invalid test cases
- Assertion-based validation

##  Tech Stack
- Python
- Selenium WebDriver

##  How to Run
1. Install dependencies:
   pip install selenium

2. Run script:
   python test.py
